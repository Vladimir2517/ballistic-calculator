# Настройка атмосферы и условий стрельбы

## Зачем это нужно

Плотность воздуха влияет на сопротивление, а значит на падение, скорость и время полета.

## Параметры атмосферы

- `--temp` температура, °C
- `--pressure` давление, гПа
- `--humidity` влажность, %
- `--altitude` высота, м

## Пример

```bash
python main.py table \
  --preset ".338 Lapua 250gr" \
  --zero 300 --range 1500 --step 100 \
  --temp 5 --pressure 980 --humidity 60 --altitude 500 \
  --wind 8 --wind-dir 270 --heading 0 --g7
```

## Практика

- Для long-range почти всегда полезно сравнить две таблицы: стандартная атмосфера vs фактические условия.
- Изменение давления и температуры заметно меняет траекторию на дальних дистанциях.
