# Пользовательский боеприпас

## Когда использовать

Если нужного пресета нет в списке или вы хотите сравнить реальные данные вашего патрона.

## Обязательные параметры

- `--mass` масса пули, г
- `--diameter` калибр, мм
- `--bc` баллистический коэффициент G1
- `--vel` начальная скорость, м/с

## Пример

```bash
python main.py custom \
  --mass 11.34 --diameter 7.82 --bc 0.505 --vel 805 \
  --zero 100 --range 900 --step 50 --wind 3
```

## Примечания

- Для boat-tail пуль можно задавать `--bc-g7` и включать `--g7`.
- Если BC неизвестен точно, начните с паспортного и откалибруйте по фактическому падению на известной дистанции.
