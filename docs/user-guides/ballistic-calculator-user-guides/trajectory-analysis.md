# Полная траектория и интерпретация результатов

## Команда

```bash
python main.py traj --preset "5.56x45 M855" --zero 100 --range 700 --step 50
```

## Что выводится

- `Дист, м`: дистанция
- `Висота, м`: вертикальная координата
- `Знос, м`: боковое отклонение
- `Vx, м/с`, `Vy, м/с`: компоненты скорости
- `V заг, м/с`: модуль скорости
- `Mach`: число Маха

## Для чего полезно

- Анализ перехода через трансзвуковой диапазон.
- Оценка запаса скорости и энергии на дистанции.
- Проверка реалистичности поведения модели при изменении атмосферы и ветра.
