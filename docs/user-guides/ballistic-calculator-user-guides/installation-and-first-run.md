# Установка и первый запуск

## Требования

- Python 3.10+
- Git (опционально, для работы с репозиторием)

## Установка

```bash
pip install -r requirements.txt
```

## Проверка CLI

```bash
python main.py presets
```

Ожидаемый результат: вывод списка доступных пресетов боеприпасов.

## Минимальная проверка расчета

```bash
python main.py table --preset "7.62x54R B32" --zero 300 --range 800 --step 100 --wind 5
```

Ожидаемый результат: таблица с колонками дистанции, падения, сноса ветром, скорости, энергии, времени полета и Mach.
