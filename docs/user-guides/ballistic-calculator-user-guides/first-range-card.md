# Быстрый старт: первая баллистическая таблица

## Задача

Построить рабочую таблицу поправок для пресета без ручной настройки физической модели.

## Команда

```bash
python main.py table --preset ".308 Win 175gr Sierra" --zero 200 --range 1000 --step 50 --wind 4
```

## Что означают ключевые параметры

- `--preset`: готовый боеприпас
- `--zero`: дистанция пристрелки
- `--range`: максимальная дистанция таблицы
- `--step`: шаг строк таблицы
- `--wind`: скорость поперечного ветра

## Интерпретация результата

- `Падение, мм`: вертикальное отклонение относительно нуля
- `Знос вітром, мм`: боковое отклонение от ветра
- `Швидк, м/с`: остаточная скорость
- `Енергія, Дж`: кинетическая энергия
- `ЧПС, с`: время полета
- `Mach`: число Маха
