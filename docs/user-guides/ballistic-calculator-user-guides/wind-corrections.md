# Поправки на ветер (мрад, МОА, клики)

## Режимы работы с ветром

- Простой: `--wind` как полный поперечный ветер.
- Расширенный: `--wind` + `--wind-dir` + `--heading` для реальной геометрии направления.

## Пример

```bash
python main.py table --preset "7.62x54R B32" --zero 300 --range 1000 --step 100 --wind 6 --wind-dir 250 --heading 40
```

## Что выводится дополнительно

После основной таблицы показывается таблица поправок:

- `мрад`: угловая поправка
- `МОА`: эквивалент в MOA
- `Кліки`: для барабана 1/4 MOA

## Рекомендация

Сохраняйте рабочие наборы параметров ветра для типовых сценариев (день/полигон/сектор), чтобы быстро повторять расчеты.
